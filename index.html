上海房金所诚聘

公司简介

房金所，即房地产金融交易所，由房金所金融服务股分有限公司运营，立足上海、深圳前海金融前沿，公司为全国第一家专注于房地产行业的金融服务公司。

此次招聘的是下属的上海房金所金融信息服务有限公司，我们致力于解决房地产金融领域的信息不对称，增强房地产的金融属性，让不动产动起来！

我们要建设的是一个房地产金融信息交流平台，并不直接从事投融资活动，亦不从中收费，与时下热闹的各类 P2P/P2C 在线金融业务有本质不同；公司在银行及房地产金融行业有很深的渊源，覆盖的投融资活动也主要发生在风控严格的“传统”信贷领域，是接触金融行业难得的机会。

前端工程师 2人

负责开发网站的 WEB 用户界面，运用先进、符合标准的技术创造直观、友好的使用界面，让用户随时随地利用各种设备都能便捷、高效的完成工作。

  • 扎实的 HTML5、现代 JavaScript、CSS3 知识。

  • 熟练使用 jQuery。
  • 熟悉任何一种现代 MVV* 框架，要有实践经验，并能根据项目需要挑选框架。
  • 有团队精神，易于沟通。

  • 能够很好地实现资深设计师制定的界面与交互效果。(我们的设计师近15年经验且会写程序，不用担心难沟通)

  • 熟悉 WebSocket、SocketIO 等有加分。

  • 熟悉 WebRTC 有加分。
  • 熟悉 Grunt 有加分。
  • 熟悉 Node.js 有加分。
  • 熟悉 Apache Cordova 有加分。

  • 熟悉微信应用开发有加分。

  • 习惯阅读英文技术资料有加分。

  • 有活跃 Github 帐号有加分。
